# BlissHashItems

## Installation

Add this line to your application's Gemfile:

    gem 'bliss_hash_items'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bliss_hash_items

## Usage

```ruby
class YourReceiptClass
  include BlissHashItems
end
```
